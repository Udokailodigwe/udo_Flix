HTTP Method Request URL Request body data format Response body data format
GET Get all movie lists '/movies' None Get data array of all movie lists in JSON format
GET Get data about a single movie title '/movies/:title' None Get an array of a single movie data in JSON format, based on title entered in the URL parameter. Example: { title: 'Lupin', director: ['Louis Leterrier', 'Marcela Said'], genre: ['Crime', 'Thriller', 'Mystery', 'Drama'], releaseYear: 2021 }
GET Get movie by genre '/movies/genre/:title' None Get an array of movie by genre in JSON format
GET Get data about a movie director '/movies/director/:name' None Get an array of a movie director data in JSON format, based on the director entered in the URL parameter
POST Allow clients to create account '/clients' Clients' user information added to the server in a form style A text message of the client's added information, included with a unique id and an extra successproof message
PUT Allow clients to update their client info '/clients/:clientsId/clientsInfo' None A text is returned alongside the updated data
POST Allow clients to add their favourite movies to their library '/clients/:clientsId/addFavourite' An array of JSON data containing the added movie. Example: { title: 'Lupin', director: ['Louis Leterrier', 'Marcela Said'], genre: ['Crime', 'Thriller', 'Mystery', 'Drama'], releaseYear: 2021 } An array of the added movie data in JSON format, with a text confirming added data and an added ID. Example: { id: 79, title: 'Lupin', director: ['Louis Leterrier', 'Marcela Said'], genre: ['Crime', 'Thriller', 'Mystery', 'Drama'], releaseYear: 2021 }
DELETE Allow clients to remove a movie from their list of favorites '/clients/:clientsId/removeFavourite' None Returns a text confirming which movie was removed
DELETE Allow clients to terminate account '/clients' None Returns a text confirming account termination
